# 🎤 אפליקציית שליטה קולית - Voice Remote

אפליקציית שליטה קולית מתקדמת המאפשרת שליטה במכשירים שונים באמצעות קול, עם תמיכה ב-IR, WiFi ו-Bluetooth.

## ✨ תכונות

- 🎤 **זיהוי קול בעברית** - שליטה מלאה באמצעות פקודות קוליות
- 📱 **תמיכה במכשירים מרובים** - ניהול מספר מכשירים במקביל
- 🔍 **סריקת IR** - העתקת כפתורים משלטי אינפרא אדום
- 📶 **תמיכה ב-WiFi** - שליטה במכשירים מחוברים לרשת
- 🔵 **תמיכה ב-Bluetooth** - חיבור למכשירים דרך Bluetooth
- 🔌 **תמיכה ב-USB** - חיבור למכשיר IR דרך USB (WebUSB API)
- 📱 **סריקת QR Code** - חיבור למכשירים דרך QR Code
- 🤖 **זיהוי אוטומטי** - זיהוי אוטומטי של מכשירים ברשת והוספה אוטומטית לרשימה
- 🔢 **קוד מספרי** - חיבור דרך קוד מספרי
- 📲 **תמיכה ב-NFC** - חיבור דרך NFC (בדפדפנים תומכים)
- 🎬 **סצנות קבועות** - שמירה והפעלה של סצנות עם מספר מכשירים
- 💾 **שמירה מקומית** - כל ההגדרות נשמרות בדפדפן

## 🚀 התקנה והפעלה

### דרך GitHub Pages

1. העלה את הקבצים ל-GitHub repository (ראה `UPLOAD_INSTRUCTIONS.md`)
2. עבור ל-Settings > Pages
3. תחת "Source" בחר Branch: **main** (או **master**)
4. Folder: **/ (root)**
5. לחץ Save
6. האפליקציה תהיה זמינה בכתובת: `https://shaharprod.github.io/voice-remote`

### הפעלה מקומית

1. פתח את `index.html` בדפדפן
2. או הפעל שרת מקומי:
```bash
python -m http.server 8000
```
ואז פתח: `http://localhost:8000`

## 📖 שימוש

### הוספת מכשיר

1. לחץ על "הוסף מכשיר"
2. מלא את הפרטים:
   - שם המכשיר
   - סוג מכשיר (טלוויזיה, מזגן, וכו')
   - סוג התחברות (IR, WiFi, Bluetooth)
   - כתובת IP (אם WiFi)

### חיבור USB

1. לחץ על הטאב "USB" בהתחברויות
2. לחץ על "התחבר למכשיר USB"
3. בחר את מכשיר ה-IR שלך מהרשימה
4. המכשיר יתחבר ויוצג פרטי המכשיר
5. כעת אפשר לשלוח פקודות IR דרך USB

### סריקת QR Code

1. לחץ על הטאב "QR Code" בהתחברויות
2. לחץ על "סרוק QR Code"
3. אפשר גישה למצלמה
4. כוון את המצלמה ל-QR Code
5. הקוד יסרק ויתחבר אוטומטית
6. או הזן קוד QR ידנית

### זיהוי אוטומטי

1. לחץ על הטאב "זיהוי אוטומטי" בהתחברויות
2. לחץ על "התחל זיהוי אוטומטי"
3. המערכת תסרוק את הרשת המקומית
4. מכשירים מזוהים יוצגו ברשימה
5. לחץ על "הוסף" כדי להוסיף מכשיר

### חיבור דרך NFC

1. בחר "NFC" כסוג התחברות
2. ודא שהדפדפן תומך ב-NFC (Chrome Android)
3. כוון את המכשיר ל-NFC tag
4. המכשיר יתחבר אוטומטית

### סריקה אוטומטית והוספה אוטומטית

1. לחץ על "סריקה אוטומטית" בקטע המכשירים
2. המערכת תסרוק את הרשת המקומית
3. מכשירים מזוהים יתווספו אוטומטית לרשימה
4. או השתמש בטאב "זיהוי אוטומטי" להוספה ידנית

### סצנות קבועות

1. לחץ על "צור סצנה חדשה"
2. הזן שם ותיאור לסצנה (למשל: "סלון", "שינה")
3. בחר מכשירים לסצנה
4. שמור את הסצנה
5. הפעל סצנה בלחיצה על "הפעל" או בקול: "הפעל סצנה [שם]"

### סריקת IR

1. בחר מכשיר IR
2. לחץ על "התחל סריקה"
3. לחץ על כפתור בשלט שלך
4. לחץ על הכפתור המתאים בממשק
5. הכפתור יישמר ויהיה זמין לשימוש

### פקודות קוליות

לחץ על "התחל האזנה" ואז אמור:

- **עוצמה**: "הגבר", "הנמך", "עצור"
- **ערוצים**: "ערוץ 5", "ערוץ הבא", "ערוץ קודם"
- **מזגן**: "הדלק מזגן", "כבה מזגן", "טמפרטורה 24"
- **תאורה**: "הדלק אור", "כבה אור"
- **כללי**: "הדלק [מכשיר]", "כבה [מכשיר]"

## 🔧 דרישות

- דפדפן מודרני התומך ב-Web Speech API (Chrome, Edge)
- מיקרופון פעיל
- עבור IR: מכשיר IR חיצוני (USB או Bluetooth)
- עבור WiFi: מכשירים תומכי WiFi עם API
- עבור Bluetooth: דפדפן תומך Web Bluetooth API
- עבור USB: דפדפן תומך WebUSB API (Chrome, Edge) ומכשיר IR עם USB
- עבור QR Code: מצלמה בגישה למצלמה
- עבור NFC: דפדפן תומך Web NFC API (Chrome Android)

## 📝 הערות טכניות

- הזיהוי הקולי משתמש ב-Web Speech API של הדפדפן
- כל הנתונים נשמרים ב-localStorage
- סריקת IR דורשת חומרה חיצונית (לא זמין בדפדפן בלבד)
- WiFi ו-Bluetooth דורשים תמיכה בדפדפן והרשאות מתאימות
