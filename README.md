# 🎤 אפליקציית שליטה קולית - Voice Remote

אפליקציית שליטה קולית מתקדמת המאפשרת שליטה במכשירים שונים באמצעות קול, עם תמיכה ב-IR, WiFi ו-Bluetooth.

## ✨ תכונות

- 🎤 **זיהוי קול בעברית** - שליטה מלאה באמצעות פקודות קוליות
- 📱 **תמיכה במכשירים מרובים** - ניהול מספר מכשירים במקביל
- 🔍 **סריקת IR** - העתקת כפתורים משלטי אינפרא אדום
- 📶 **תמיכה ב-WiFi** - שליטה במכשירים מחוברים לרשת
- 🔵 **תמיכה ב-Bluetooth** - חיבור למכשירים דרך Bluetooth
- 💾 **שמירה מקומית** - כל ההגדרות נשמרות בדפדפן

## 🚀 התקנה והפעלה

### דרך GitHub Pages

1. העלה את הקבצים ל-GitHub repository
2. עבור ל-Settings > Pages
3. בחר branch ו-folder
4. האפליקציה תהיה זמינה בכתובת: `https://[username].github.io/[repository-name]`

### הפעלה מקומית

1. פתח את `index.html` בדפדפן
2. או הפעל שרת מקומי:
```bash
python -m http.server 8000
```
ואז פתח: `http://localhost:8000`

## 📖 שימוש

### הוספת מכשיר

1. לחץ על "הוסף מכשיר"
2. מלא את הפרטים:
   - שם המכשיר
   - סוג מכשיר (טלוויזיה, מזגן, וכו')
   - סוג התחברות (IR, WiFi, Bluetooth)
   - כתובת IP (אם WiFi)

### סריקת IR

1. בחר מכשיר IR
2. לחץ על "התחל סריקה"
3. לחץ על כפתור בשלט שלך
4. לחץ על הכפתור המתאים בממשק
5. הכפתור יישמר ויהיה זמין לשימוש

### פקודות קוליות

לחץ על "התחל האזנה" ואז אמור:

- **עוצמה**: "הגבר", "הנמך", "עצור"
- **ערוצים**: "ערוץ 5", "ערוץ הבא", "ערוץ קודם"
- **מזגן**: "הדלק מזגן", "כבה מזגן", "טמפרטורה 24"
- **תאורה**: "הדלק אור", "כבה אור"
- **כללי**: "הדלק [מכשיר]", "כבה [מכשיר]"

## 🔧 דרישות

- דפדפן מודרני התומך ב-Web Speech API (Chrome, Edge)
- מיקרופון פעיל
- עבור IR: מכשיר IR חיצוני (USB או Bluetooth)
- עבור WiFi: מכשירים תומכי WiFi עם API
- עבור Bluetooth: דפדפן תומך Web Bluetooth API

## 📝 הערות טכניות

- הזיהוי הקולי משתמש ב-Web Speech API של הדפדפן
- כל הנתונים נשמרים ב-localStorage
- סריקת IR דורשת חומרה חיצונית (לא זמין בדפדפן בלבד)
- WiFi ו-Bluetooth דורשים תמיכה בדפדפן והרשאות מתאימות
